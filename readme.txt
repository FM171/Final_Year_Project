Steps to run wesbtie 

please note Python version 3.9 needs to be installed and open for the wesbite to run

1. change the directory in comman prompt to be in the website folder 
2. enter this commad : env\Scripts\activate
3. enter this command: set FLASK_APP=run.py 
4. enter this command: flask run
5. You should see the following message "Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)"
6. copy and paste http://127.0.0.1:5000/ into the google url search bar and hit enter


If the env folder has not been copied please enter these commands first 

1.change the directory in command prompt to be in the projectfolder 
2. enter this commad : py -3 -m venv env
3. enter this commad : env\Scripts\activate
4. enter this commad : pip install -r requirements.txt
5. enter this command: set FLASK_APP=run.py 
6. enter this command: flask run
6. You should see the following message "Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)"
7. copy and paste http://127.0.0.1:5000/ into the google url search bar and hit enter
